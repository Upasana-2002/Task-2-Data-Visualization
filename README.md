# Task 2 – Data Visualization & Storytelling  
This repository contains the Power BI dashboard and insights created for **Task 2** of the Data Analyst Internship.  
The objective is to analyze the Superstore dataset (2014–2017) and present meaningful business insights using charts.

---

## Tools Used
- **Power BI** 
- **Microsoft Excel** 
- **GitHub** 

---

## Objective
To create a clear and compelling **visual story** from the Superstore dataset using interactive dashboards and 7 different charts:
1. Sales by Category  
2. Profit by Sub-Category  
3. Monthly Sales Trend (2014–2017)  
4. Sales by Region  
5. Discount vs Profit  
6. Sales by Ship Mode  
7. Top N Categories by Quantity Sold  

A **Year slicer (2014–2017)** is included to view trends year-wise.

---

## Key Insights (2014–2017)
- Technology has the highest sales, while Office Supplies is lowest until 2016 and Furniture is lowest in 2017.
- Phones, Copiers, and Accessories give highest profit, whereas Furnishings and Appliances show losses.
- 2017 has the highest sales overall, with sales peaking in Nov–Dec and remaining lowest in Jan–Mar.
- West region leads sales every year, outperforming all other regions.
- Higher discounts reduce profit, with many high-discount orders turning into losses.
- Standard Class is the most used shipping mode across all four years.
- Office Supplies has the highest quantity sold, while Technology has the lowest.
